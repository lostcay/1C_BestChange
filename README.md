# Полностью автономное расширение для работы с сервисом BestChange. 
Поможет выбрать лучший обменник для балансов криптобирж, денежных переводов, интернет-банкинга, криптовалюты, а так же наличных и электронных денег.

Технические требования:
- Платформой версии 8.3.14 и выше

После загрузки расширения необходимо убрать «Безопасный режим» и «Защиту от опасных действий», затем необходимо перезайти в программу.

После перезагрузки должно пройти от минуты до двух, для первоначальной настройки и получения первичных данных.

Расширение добавляет единственную подсистему «BestChange». Основной функционал содержится во вкладке «Главная страница».

После установки направления обмена в табличной части, ниже, отобразится список наиболее выгодных предложений от разных обменников.

При выборе наличных денег в качестве меняемой и/или получаемой валюты, появится возможность выбрать город, в котором находится обменный пункт, если оставить поле пустым отобразятся все варианты с указанием города.

Во вкладке «Дополнительно» можно воспользоваться калькулятором для расчета необходимой суммы.

Так же реализована технология автоматического отслеживания необходимого курса. Во вкладке дополнительно на странице «Оповещения» можно создать заявку. Когда цена будет удовлетворять заявленную, пользователю высветится оповещение.

Нажав на звездочку справа от направления обмена, оно добавится/удалится из избранных. По нажатии на кнопку командной панели «Выбрать из избранных», отобразится список избранных, двойным щелчком можно выбрать необходимое направление.

На форме оповещения можно просмотреть подходящие предложения обменников. Если заявка более неактуальна, нажмете кнопку «Удалить заявку».

На данный момент справочная информация несет исключительно технический характер

Реализовано 3 роли:

- ОсновнаяРоль – устанавливается абсолютно всем пользователям, из-за ограничений платформы в отношении технологии расширений

- Пользователь – может пользоваться описанным выше функционалом

- Администратор – имеет права на:

  - Изменение настроек расширения (описывать все нет смысла, все интуитивно понятны)
  - Просмотр журнала обновлений данных
  - Просмотр всех заявок на оповещения
  - Просмотр избранных направлений обмена всех пользователей
  - Просмотр истории переходов к обменникам
  - Просмотр истории поиска направлений обмена
  - Для непредвиденных ситуаций во вкладке «Обслуживание» предусмотрены средства для ручного исправления поврежденных данных
  - Доступ ко вкладке «Интернет-поддержка», где можно проверить наличие новой версии расширения, а также получить актуальную версию и инструкцию к обновлению

Расширение полностью бесплатно. Исходный код открыт. Об ошибках и пожеланиях по доработке можете писать в issues.
